# 无模板引导提示功能实现

> 用于多页面无模板时的用户引导功能设计文档
> 版本：1.0 | 创建日期：2025-02-28

---

## 一、功能概述

当本地没有对应类型的模板时，显示引导提示框，引导用户前往"模板仓库"页面下载模板。

### 涉及的页面

| 页面 | 模板类型 | 跳转目标 |
|------|---------|---------|
| 预定义场景 (Cases) | Terraform 场景模板 | registry |
| 专项模块 (SpecialModules) | Userdata 模板 | registry |
| 编排管理 (Compose) | Compose 模板 | registry |

---

## 二、实现逻辑

### 2.1 组件通信设计

由于页面组件是在 App.svelte 中通过条件渲染引入的，需要使用 props 传递 `onTabChange` 回调函数来实现页面跳转。

**App.svelte 中的组件调用方式：**

```svelte
<!-- 预定义场景 -->
<Cases {t} onTabChange={(tab) => activeTab = tab} />

<!-- 专项模块 -->
<SpecialModules {t} onTabChange={(tab) => activeTab = tab} />

<!-- 编排管理 -->
<Compose {t} onTabChange={(tab) => activeTab = tab} />
```

### 2.2 组件接收 props

```svelte
// SpecialModules.svelte / Compose.svelte 中
let { t, onTabChange } = $props();
```

### 2.3 条件渲染逻辑

**Cases.svelte:**
```svelte
{#if !templates || templates.length === 0}
  <div class="bg-blue-50 border border-blue-100 rounded-xl p-5 mb-4">
    <!-- 提示内容 -->
    <button onclick={() => onTabChange && onTabChange('registry')}>
      {t.noTemplatesHintButton}
    </button>
  </div>
{/if}
```

**SpecialModules.svelte:**
```svelte
<!-- Vulhub 子页面 -->
{#if templates.length === 0}
  <div class="bg-blue-50 border border-blue-100 rounded-xl p-5 mb-4">
    <!-- 提示内容 -->
  </div>
{/if}

<!-- C2 子页面 -->
{#if templates.length === 0}
  <div class="bg-blue-50 border border-blue-100 rounded-xl p-5 mb-4">
    <!-- 提示内容 -->
  </div>
{/if}

<!-- AI 子页面 -->
{#if templates.length === 0}
  <div class="bg-blue-50 border border-blue-100 rounded-xl p-5 mb-4">
    <!-- 提示内容 -->
  </div>
{/if}
```

**Compose.svelte:**
```svelte
{#if !templatesLoading && composeTemplates.length === 0}
  <div class="bg-blue-50 border border-blue-100 rounded-xl p-5">
    <!-- 提示内容 -->
  </div>
{/if}
```

## 四、UI 设计规范

### 4.1 样式规范

统一使用蓝色主题样式，与预定义场景保持一致：

```html
<div class="bg-blue-50 border border-blue-100 rounded-xl p-5 mb-4">
  <div class="flex items-start gap-3">
    <!-- 图标 -->
    <svg class="w-5 h-5 text-blue-500 flex-shrink-0 mt-0.5" ...></svg>
    <div class="flex-1">
      <!-- 提示文字 -->
      <p class="text-[13px] text-blue-700">{t.xxxHint}</p>
      <!-- 跳转按钮 -->
      <button class="mt-3 h-8 px-4 bg-blue-500 text-white text-[12px] font-medium rounded-lg hover:bg-blue-600 transition-colors cursor-pointer">
        {t.xxxHintButton}
      </button>
    </div>
  </div>
</div>
```

### 4.2 设计要点

- **背景色**: `bg-blue-50`
- **边框色**: `border border-blue-100`
- **圆角**: `rounded-xl`
- **内边距**: `p-5`
- **下边距**: `mb-4` (用于与下方内容分开)
- **图标**: 使用 Heroicons 的信息图标 (信息提示)
- **按钮**: 蓝色实心按钮，与主题色一致

## 六、注意事项

### 6.1 加载状态处理

在显示无模板提示之前，需要先判断模板是否加载完成：

```svelte
<!-- 正确：等待加载完成后再判断 -->
{#if !templatesLoading && templates.length === 0}
  <!-- 显示提示 -->
{/if}

<!-- 错误：未判断加载状态 -->
{#if templates.length === 0}
  <!-- 加载中时也会显示 -->
{/if}
```

### 6.2 动画冲突问题

如果在页面切换动画中使用了 `transform` 属性，会导致 fixed 定位的对话框定位错误（相对于动画容器而非视口）。

**解决方案：** 页面切换动画只使用 opacity，不使用 transform：

```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```
