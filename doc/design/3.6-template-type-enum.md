# 模板类型枚举重构

## 概述

将模板类型从多个布尔字段改为单一枚举值，统一管理不同类型的模板。

---

## 1. 模板类型定义

### 1.1 类型枚举

**文件**: `mod/tmpl.go`

```go
// TemplateType 模板类型枚举
type TemplateType string

const (
	TemplateTypePreset   TemplateType = "preset"   // 预定义模板（场景管理使用）
	TemplateTypeBase     TemplateType = "base"     // 自定义部署模板
	TemplateTypeUserdata TemplateType = "userdata" // Userdata 模板
	TemplateTypeCompose  TemplateType = "compose"  // Compose 模板
)
```

### 1.2 模板结构体

**文件**: `mod/tmpl.go`

```go
// RedcTmpl 对应本地 case.json 的结构
type RedcTmpl struct {
	Name         string       `json:"name"`
	Description  string       `json:"description"`
	User         string       `json:"user"`
	Version      string       `json:"version"`
	RedcModule   string       `json:"redc_module"`
	TemplateType TemplateType `json:"template"` // 使用枚举替代布尔字段
	Path         string       `json:"-"`
}
```

## 2. 解析逻辑修改

### 2.1 caseJSON 结构体

**文件**: `mod/template_manager.go`

```go
// caseJSON 用于解析 case.json 文件的结构
type caseJSON struct {
	Name               string       `json:"name"`
	Description        string       `json:"description"`
	User               string       `json:"user"`
	Version            string       `json:"version"`
	RedcModule         string       `json:"redc_module"`
	TemplateType       TemplateType `json:"template"` // 改为枚举
	Provider           string       `json:"provider"`
	SupportedProviders []string     `json:"supported_providers"`
}
```

### 2.2 判断基础模板

```go
// IsBaseTemplate 判断是否为基础模板
func (m *TemplateManager) IsBaseTemplate(templatePath string) (bool, error) {
	caseFilePath := filepath.Join(templatePath, TmplCaseFile)
	data, err := os.ReadFile(caseFilePath)
	if err != nil {
		return false, err
	}

	var caseData caseJSON
	if err := json.Unmarshal(data, &caseData); err != nil {
		return false, err
	}

	// 使用枚举判断
	return caseData.TemplateType == TemplateTypeBase, nil
}
```

### 2.3 过滤预定义模板

**文件**: `mod/tmpl.go`

```go
// ListTemplates 返回预定义模板列表（过滤掉自定义模板等）
func ListTemplates() ([]*RedcTmpl, error) {
	// ... 扫描逻辑 ...

	for _, dirPath := range dirs {
		t, err := readTemplateMeta(dirPath)
		// 过滤掉非预定义模板
		if t.TemplateType == TemplateTypeBase ||
			t.TemplateType == TemplateTypeUserdata ||
			t.TemplateType == TemplateTypeCompose {
			continue
		}
		templates = append(templates, t)
	}
	return templates, nil
}
```

## 6. 迁移说明

### 旧的布尔字段格式（已废弃）

```json
{
  "is_base_template": true,
  "is_userdata_template": true,
  "is_compose_template": true
}
```

### 新的枚举格式

```json
{
  "template": "base"
}
```

或

```json
{
  "template": "userdata"
}
```

或

```json
{
  "template": "compose"
}
```

### 预定义模板

预定义模板不需要添加 `template` 字段，系统会根据是否存在于特定目录来判断类型。
